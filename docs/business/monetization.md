## 1. Принципы Монетизации

| Принцип | Описание |
|---------|----------|
| **🔑 NO COMMISSION** | Мы НЕ берём комиссию от сделок — это главное USP (Unique Value Proposition), отличающее нас от конкурентов. Подробнее в [Lean Canvas](lean-canvas.md) |
| **Value-Based** | Ценообразование основано на ценности для клиента, а не на себестоимости |
| **Freemium** | Базовый функционал — бесплатно; платные услуги — для роста бизнеса |
| **Predictable Revenue** | Подписки и фиксированные платежи обеспечивают предсказуемый доход |

## 2. Источники Дохода (7 позиций)

### 2.1. Premium-тарифы для продавцов

| Параметр | Значение |
|----------|----------|
| **Что это** | Пакеты с расширенным функционалом: приоритетный matching, расширенная аналитика, API-доступ |
| **Кому нужно** | Активным продавцам, которые хотят больше входящих запросов и автоматизации |
| **Цена** | Базовый: бесплатно; Оптимальный: 15 000 ₽/мес.; Максимальный: 50 000 ₽/мес. |
| **Модель** | Подписка (ежемесячно) |

**Фичи по тарифам:**

| Фича | Базовый | Оптимальный | Максимальный |
|------|---------|-------------|--------------|
| Размещение офферов | ✓ | ✓ | ✓ |
| Базовая аналитика | — | 50 запросов/мес | ∞ |
| Приоритетный matching | — | +20% к видимости | +50% к видимости |
| API-доступ | — | — | ✓ |
| Персональный менеджер | — | — | ✓ |
| White-label / брендирование | — | — | ✓ |

| Параметр | Значение |

## 3. Приоритеты Запуска

### 🟢 MVP (Запуск в первую очередь)

| Услуга | Обоснование |
|--------|-------------|
| **Premium-тарифы для продавцов** | Основной revenue driver; простая реализация; понятная ценность |
| **Продвижение в поиске (Featured Listings)** | Высокий спрос (по данным Industrial.Market); быстрый доход |
| **Базовая аналитика** | Встроена в Premium-тариф; низкая стоимость разработки |

### 🟡 Фаза 2 (После MVP, 3-6 месяцев)

| Услуга | Обоснование |
|--------|-------------|
| **Верификация и сертификация** | Требует ручной работы / партнёрств; высокая ценность для рынка |
| **AI-консультация и Premium Matching** | Нужна база данных и процессы; ВИП-сегмент |
| **Создание карточек товаров** | Контент-команда; масштабируемо |

### 🔵 Фаза 3 (После масштабирования, 6-12 месяцев)

| Услуга | Обоснование |
|--------|-------------|
| **API-доступ** | Нужен зрелый функционал; интерес крупным игрокам |
| **Продвинутая аналитика** | Большие данные; ML-модели |
