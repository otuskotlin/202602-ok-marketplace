# Value Proposition — Ценностное Предложение

**Продукт:** B2B Маркетплейс Промышленных Компонентов  
**Версия:** 1.0  
**Статус:** Ready  
**Дата:** 2026-03-25

---

## 1. USP (Unique Selling Proposition)

| Компонент | Описание |
|-----------|----------|
| **Специализация** | Единственная платформа, сфокусированная исключительно на промышленных компонентах: моторы, электронные компоненты, датчики, ПЛК, приводы |
| **Smart Matching** | AI-алгоритм, который автоматически сводит запросы покупателей с релевантными предложениями, сокращая цикл закупки с типичных 14+ дней до 2-3 дней |
| **No Logistics** | Мы не храним товары, не занимаемся доставкой — только информационный обмен и верификация сделок. Это снижает CAPEX и позволяет масштабироваться быстрее |
| **Верификация продавцов** | Проверка сертификатов, рейтинговая система, история сделок — критически важно для промышленных закупок |
| **Прозрачность цен** | Рыночные цены в открытом доступе — пользователь (в роли покупателя) видит честные предложения без «накруток» посредников |

## 5. Pain Relievers (Устранители болей)

| Боль | Как решаем |
|------|------------|
| **Закрытый рынок** | AI-matching даёт доступ к запросам, которые ищут именно этот товар |
| **Холодные звонки** | Пользователь (в роли покупателя) сам находит продавца — без обзвона |
| **Нет visibility** | Каталог + SEO + рейтинг = клиент находит сам |
| **Поиск = кошмар** | Единая база с фильтрами по категории, цене, наличию |
| **Нет рейтинга** | Верификация + отзывы + история сделок |
| **Посредники** | No commission — пользователь (в роли продавца) работает напрямую |

## 6. Product-Market Fit

| Фича | Ценность | Боль клиента | Приоритет |
|------|----------|--------------|-----------|
| AI-Matching | 🔴 High (0.8) | 🔴 High (0.9) | **Приоритет #1** |
| No Commission | 🔴 High (0.9) | 🟡 Medium (0.7) | **Приоритет #2** |
| Верификация | 🟡 Medium (0.7) | 🔴 High (0.8) | **Приоритет #3** |
| Специализация | 🟡 Medium (0.6) | 🟡 Medium (0.6) | Усилить |
| Прозрачность цен | 🟡 Medium (0.7) | 🟢 Low (0.5) | Конкурентное преимущество |
| Каталог | 🟢 Low (0.5) | 🟢 Low (0.4) | Низкий приоритет |

### Interpretation

- **Приоритет #1-3**: Must-have фичи для MVP
- **Усилить**: Нужно развивать для усиления позиции
- **Конкурентное преимущество**: Дифференциатор vs конкуренты
- **Низкий приоритет**: Roadmap, не для MVP
