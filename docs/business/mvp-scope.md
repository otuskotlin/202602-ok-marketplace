## Ключевое допущение

**Вместо UI-чата — email-коммуникация:**

- Переговоры ведутся через **email** (публикация контактов в объявлениях)
- Matching — **on-demand** в UI при просмотре "Моих объявлений"

**Почему:**
- UI-чат требует WebSocket инфраструктуру — дорого для MVP
- Email — уже есть (SMTP), интегрируется в любой почтовый клиент
- B2B-пользователи привыкли к email-переписке
- **Email-matching = спам-риск → НЕ используется**
