# REQ-005: Каталог и поиск

## Business Value
Обеспечить удобный поиск и навигацию по каталогу промышленных компонентов с фильтрами и сортировкой.

## User Stories

**US-013: Поиск по ключевым словам**
* **As a** Пользователь (в роли покупателя)
* **I want** Искать товары по названию
* **So that** Найти нужный товар быстро

### Acceptance Criteria (BDD)
* **Given:** Пользователь вводит запрос в строку поиска
* **When:** Нажимает "Найти"
* **Then:** Система отображает релевантные OFFER с подсветкой совпадений

**US-014: Фильтрация по категориям**
* **As a** Пользователь (в роли покупателя)
* **I want** Фильтровать товары по категориям
* **So that** Сузить выборку до нужных товаров

### Acceptance Criteria (BDD)
* **Given:** Пользователь находится в каталоге
* **When:** Выбирает категорию "ПЛК" и подкатегорию "Siemens"
* **Then:** Система показывает только OFFER выбранной категории

**US-015: Сортировка результатов**
* **As a** Пользователь (в роли покупателя)
* **I want** Сортировать результаты поиска
* **So that** Найти товары по нужному критерию

### Acceptance Criteria (BDD)
* **Given:** Пользователь видит результаты поиска
* **When:** Выбирает сортировку "Цена по возрастанию"
* **Then:** Система пересортировывает результаты

## System Constraints

| Constraint | Value | Notes |
|-----------|-------|-------|
| Latency (search) | < 300 ms | Время поиска |
| Latency (filter) | < 200 ms | Применение фильтров |
| Throughput | 5 req/sec | Пиковая нагрузка Year 2 |
| Availability | 99.5% | SLA для B2B платформы |
| Max results | 1000 | Пагинация обязательна |

## Out of Scope
- Голосовой поиск
- Визуальный поиск по фото
- AR-примерка
- Интеграция с CAD-системами
- Персонализированная лента
