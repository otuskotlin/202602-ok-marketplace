**Статус:** Roadmap

# REQ-006: Профиль компании и рейтинг

## Business Value
Создать доверие между участниками через прозрачные профили компаний и систему рейтингов.

## User Stories

**US-016: Просмотр профиля компании**
* **As a** Пользователь
* **I want** Просмотреть информацию о компании
* **So that** Принять решение о сотрудничестве

### Acceptance Criteria (BDD)
* **Given:** Пользователь выбирает OFFER или REQUEST конкретной компании
* **When:** Нажимает на название компании
* **Then:** Система отображает профиль: название, ИНН, контакты, рейтинг, историю сделок, отзывы

**US-017: Оставление отзыва о сделке**
* **As a** Пользователь (в роли покупателя)
* **I want** Оценить продавца после сделки
* **So that** Помочь другим пользователям принять решение

### Acceptance Criteria (BDD)
* **Given:** Завершённая сделка между пользователями
* **When:** Пользователь (в роли покупателя) оценивает продавца (1-5 звёзд) и пишет комментарий
* **Then:** Отзыв сохраняется, обновляется рейтинг продавца

**US-018: Редактирование профиля компании**
* **As a** Пользователь
* **I want** Изменить информацию о компании
* **So that** Поддерживать актуальные данные профиля

### Acceptance Criteria (BDD)
* **Given:** Авторизованный пользователь
* **When:** Изменяет описание компании, контакты, логотип
* **Then:** Профиль обновляется, изменения видны другим пользователям

## System Constraints

| Constraint | Value | Notes |
|-----------|-------|-------|
| Latency (view profile) | < 200 ms | Загрузка профиля |
| Latency (rating update) | < 300 ms | Обновление рейтинга |
| Throughput | 5 req/sec | Пиковая нагрузка Year 2 |
| Availability | 99.5% | SLA для B2B платформы |
| Rating scale | 1-5 | Звёздная система |

## Out of Scope
- Верификация через ЕГРЮЛ (автоматическая)
- Кредитный рейтинг
- Финансовая отчётность
- Судебная история
- Интеграция с СПАРК
